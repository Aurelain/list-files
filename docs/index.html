List files from your local drive

Feed directories (or files) using any of the following methods:
  • Drag-and-drop from your file explorer
  • Browse for items:
  • Paste strings of absolute paths anywhere inside
  • Append "?x" to the address-bar, where "x" can be a string of absolute paths
  • Use `postMessage` to send a `payload` (string of absolute paths) to this page
